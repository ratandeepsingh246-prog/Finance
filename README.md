# Finance
Interactive Power BI Finance Dashboard analyzing sales, profit, discounts, and units sold across products, segments, countries, and time. 
Includes KPI cards, trend analysis, segment profitability, product performance, and geographic insights to support data-driven financial decisions.

## Project Overview
This project presents an interactive **Finance Dashboard** built in Power BI to analyze sales and profitability 
performance across products, segments, geographies, and time. The dashboard converts raw financial data into clear insights for monitoring trends and supporting business decisions.

## Key Metrics
- Total Sales  
- Units Sold  
- Gross Sales, Discounts, and Net Sales  
- Profit by segment and product  

## Dashboard Pages

### Page 1: Financial Overview
- KPI cards for sales and units sold  
- Sales trend by month  
- Sales by product and segment  
- Manufacturing price vs sale price comparison  

### Page 2: Performance Analysis
- Sales by country (map visualization)  
- Profit contribution by segment  
- Sales and discounts by product  
- Segment-wise comparison of sales and profit  

## Tools & Skills Used
- Power BI  
- Power Query (data transformation)  
- DAX (measures and aggregations)  
- Financial data analysis and visualization  

## Project Files
- Power BI (.pbix) file  
- Dashboard screenshots  

## Usage
Open the `.pbix` file in Power BI Desktop and use filters to explore financial performance by year, segment, product, and geography.
